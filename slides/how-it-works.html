How It Works

Three steps. Zero effort.

1

Install the GitHub App

One click to connect askmanu to your repositories. No config files, no CI changes.

2

askmanu watches your releases

When you ship a new release, askmanu reads the changes and opens an issue outlining what docs need updating.

3

Review, merge, repeat

One focused PR at a time. Merge it, and askmanu creates the next one — continuing the plan or picking up new work. Always sequential, always reviewable.
